# 🚀 Project Highlights

### 🛡️ [Automated Vulnerability Scanner](https://github.com/AtharvKashyap/Automated_Vulnerability_Scanner)

A cybersecurity automation tool for tracking recent CVEs, mapping vulnerabilities to known exploited threats, and generating analyst-friendly reports. Built to support vulnerability management workflows by combining automation, structured data processing, and security reporting.

**Focus areas:** Python automation, CVE tracking, CISA KEV enrichment, vulnerability management, Excel reporting, security operations.

---

### ⚙️ [Lucene Shared Merge Scheduler](https://github.com/AtharvKashyap/Lucene-Shared-Merge-Scheduler)

A performance and correctness-focused systems project for Apache Lucene that explores shared scheduling for merge operations. Built around improving resource coordination, testing edge cases, and reasoning about concurrency-heavy behavior in a real-world search engine codebase.

**Focus areas:** Java, systems programming, concurrency, performance engineering, correctness testing, Apache Lucene.

---

### 🤖 [PandaPad](https://github.com/AtharvKashyap/PandaPad)

An AI-powered learning companion for children designed to support curiosity, conversation, and early learning. The project explores safe AI interaction for young users through voice-driven interfaces, guardrails, memory, and child-friendly responses.

**Focus areas:** Swift/iOS, AI product development, child-safe LLM design, speech interfaces, user experience, startup building.
